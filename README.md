# AI Etsy Product Description Generator ✨

A simple, web-based AI tool that helps Etsy sellers generate **SEO-optimized product titles, descriptions, and tags** automatically using AI. 
Built for a clean, user-friendly experience with responsive design and copy-to-clipboard functionality.

Built in Features

- Generate **SEO-friendly Etsy product titles**.
- Generate **persuasive product descriptions**.
- Generate **10 relevant Etsy tags** for better search visibility.
- **Copy to clipboard** button for instant use.
- **Responsive design** works on desktop and mobile.
- **Clean and modern UI** with styled inputs and buttons.
- Fully **web-based** using Cloudflare Pages + Workers.
- Automated AI generation via **OpenAI API**.
